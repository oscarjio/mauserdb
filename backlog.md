## Förbättringar

- [x] **VD:s morgonrapport (auto-vy)** — sammanfattning av gårdagens produktion: mål vs utfall, stopp, kassation, highlights, varningar. Utskriftsvänlig
- [x] **OEE-waterfall/brygga** — visuell nedbrytning av OEE-förluster: tillgänglighet → prestanda → kvalitet. Waterfall-stapeldiagram som visar var tid förloras
- [ ] **Operatörs-onboarding tracker** — visa nya operatörers lärlingskurva: hur snabbt de når team-genomsnittet i IBC/h de första veckorna
- [ ] **Skiftöverlämningslogg** — digital överlämning mellan skift: vad gjordes, pågående problem, specialinstruktioner
- [ ] **Snabbkommandon/favoritvy** — VD kan bokmärka sina mest använda sidor för snabb åtkomst från startsidan
- [ ] **Skiftvis produktionsjämförelse** — jämför skift A/B/C: IBC/h, stopp, kassation. Vilka skift presterar bäst?
- [ ] **Drifttids-timeline** — visuell tidslinje per dag: gröna block = körning, röda = stopp, grå = ej planerat. Klickbar för detaljer
- [ ] **Kassationsorsak-drill-down** — från kassationsgrad → klicka ner till specifika orsaker → enskilda händelser. Trädvy/hierarki
